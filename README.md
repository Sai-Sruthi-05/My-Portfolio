# 🌐 Sai Sruthi Tejasvi P — Portfolio

A clean, responsive personal portfolio website built with pure HTML, CSS, and JavaScript. Features smooth scroll animations, a custom cursor, and a soft white & light blue aesthetic.

## ✨ Features

- Custom animated cursor with hover effects
- Scroll-triggered fade-in animations
- Fully responsive (mobile, tablet, desktop)
- Smooth section navigation
- Interactive project cards & skill cards
- No frameworks or build tools required — just one HTML file

## 🛠 Built With

- HTML5
- CSS3 (custom properties, flexbox, grid, keyframe animations)
- Vanilla JavaScript

## 📁 Sections

- **Hero** — Introduction, key stats, CTA buttons
- **About** — Bio and contact info cards
- **Skills** — Languages, frameworks, tools, cloud, analytics
- **Projects** — Shopify Data Insights, CodeBuddy, Fraud Risk Dashboard
- **Experience** — Business Strategy Intern at Talentship
- **Certifications** — AWS, MongoDB, Oracle, Microsoft Azure
- **Achievements** — Flipkart GRiD Top 2%, Unstop Rank 14, WiOS Lead
- **Contact** — Email, LinkedIn, GitHub
